# App Clube do Livro

App para o Clube do Livro - Colégio Monte Alto

# Sugestão inicial:

Back end: NodeJS + MongoDB

Front end: Ionic 3

Hospedagem: Azure

Funcionalidades:
- Listagem de livros
- Empréstimo e devolução
- Alerta para data de devolução do livro
- Inclusão de avaliação do livro
- Controle de acesso dos pais
